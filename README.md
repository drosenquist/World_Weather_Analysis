# World_Weather_Analysis
PlanMyTrip app provides random vacation destination suggestions to a user based on weather tempature preference (prefered minimum and maximum tempature) through the use of two APIs (Google Maps API and a weather data API). Four locations were chosen by the user and a (driving) route provided between those locations with the start and ending being the same location. The map shows hotel location in the cities chosen along with weather conditions.
